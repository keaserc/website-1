Практическое руководство по написанию базового кода

Модуль 1

Основы синтаксиса и Переменные

В Python переменная создается знаком =. Тип данных (число или текст) указывать не нужно. Имена пишутся латиницей. Запрещено использовать пробелы и начинать имя с цифры. Регистр важен (x и X — разные переменные).

x = 10          # Целое число
y = 5.5         # Дробное число
is_valid = True # Логическое значение (True/False с большой буквы)
Модуль 2

Строки текста и Вывод (print)

Любой текст оборачивается в двойные "..." или одинарные кавычки. Сложение строк (конкатенация) соединяет их без пробелов, поэтому пробелы внутри кавычек нужно ставить вручную. Функция print() выводит результат на экран.

name = "Иван"
text = "Привет, " + name + "!"
print(text) # Выведет на экран: Привет, Иван!
Модуль 3

Преобразование типов данных

Python не умеет математически складывать текст и числа. Текст из цифр нужно принудительно превратить в число функцией int(). Число переводится в текст функцией str().

num_str = "20"
num = int(num_str) # Перевели строку в число 20
total = num + 5    # Теперь можно складывать математически
print("Результат: " + str(total)) # Перевели число обратно в текст
Модуль 4

Логика условий и Отступы

Проверка равенства пишется как ==, а неравенства — как !=. В конце строки с условием if, elif или else строго ставится двоеточие :. Вложенный исполняемый код на новой строке обязательно должен иметь отступ ровно в 4 пробела.

a = 15
if a > 20:
    print("Больше 20")
elif a == 15:
    print("Равно 15")
else:
    print("Меньше 15")
Модуль 5

Работа со списками (Lists)

Список хранит набор элементов в квадратных скобках через запятую. Нумерация (индексы) начинается с 0. Функция len() считает количество элементов. Метод .append() вставляет новый элемент в самый конец списка.

items = ["меч", "щит"]
print(items[0])       # Выведет первый элемент: меч
items.append("зелье") # Добавили элемент в конец списка
print(len(items))     # Выведет общую длину списка: 3
Модуль 6

Циклы for и диапазон range()

Цикл for перебирает элементы списков по очереди. Функция range(N) автоматически генерирует последовательность чисел от 0 до N-1, задавая нужное количество повторений для цикла.

for x in ["A", "B"]:
    print("Элемент: " + x)
for i in range(3):
    print("Повторение номер: " + str(i)) # Выведет индексы 0, 1, 2
Модуль 7

Хранение пар в словарях (Dict)

Словарь хранит данные парами «ключ: значение» в фигурных скобках. Чтобы извлечь или изменить значение, вместо числового индекса в квадратных скобках указывается имя ключа латиницей или в кавычках.

user = {
    "name": "Alex",
    "level": 5
}
print(user["name"]) # Найдет значение по ключу: Alex
user["level"] = 6   # Изменили значение внутри словаря
Модуль 8

Объявление собственных функций

Функция создается словом def, за которым идет имя и круглые скобки с параметрами. Строка завершается двоеточием, тело функции пишется с отступом. Оператор return возвращает вычисленный результат обратно.

def calc(a, b):
    result = a + b
    return result # Передает ответ наружу

res = calc(10, 20) # Вызов функции, в res запишется 30
