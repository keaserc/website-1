Урок 1: Создаем переменную

В Python переменная — это именная коробка для хранения данных. Пишется имя латиницей, знак равенства = и число.

gold = 500

Задание: Найдите вариант ответа, который создаст переменную "x" со значением 10.

Урок 2: Текст и строки

Любые текстовые буквы в коде называются строками. Чтобы компьютер понимал, где текст, а где команда, слова оборачивают в кавычки.

hero = "Алекс"

Задание: Какой код правильно сохранит слово "Python" в переменную "text"?

Урок 3: Калькулятор вычислений

Python считает со скоростью света. Можно написать математический пример сразу после знака равно, и ответ сам сохранится в переменную.

points = 5 + 10

Задание: Выберите код, где переменная "num" равна результату сложения "20 + 30".

Урок 4: Печать на экран

Чтобы увидеть скрытые вычисления кода прямо на экране компьютера, используется встроенная команда вывода print().

print(77)

Задание: Выберите верный способ напечатать на экране число 100.

Урок 5: Склеивание строк

Знак плюс + умеет соединять строки текста. Если сложить два слова, они превратятся в единое предложение.

phrase = "Hi " + "Oleg"

Задание: Какая строчка сложит строки "Hi " и "User" в переменную "msg"?

Урок 6: Разветвление if

Команда if (если) проверяет условия. Для точного равенства нужен двойной знак ==, а строка должна заканчиваться двоеточием.

if status == "admin":
    print("Привет")

Задание: Укажите правильный синтаксис: если "a" равно 5, то вывести 1.
